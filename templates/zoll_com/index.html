Temperature Scarf - Project Spotlight

The Temperature Scarf project (also known as My Year In Temperatures) is a knitting aid designed to help with a scarf pattern from the knitting / crocheting site Ravelry. First posted by Kristen Cooper, the idea is to use local daily average temperatures as a color key, producing a scarf that reads like daily thermometer of your hometown.

I saw the opportunity to create an application that could automatically display the correct temperatures and colors for each day. With this functionality, the user no longer has to manually look up temperatures and find the correct color manually.

A simple color configuration

After picking colors and assigning them names for easy identification, the user can generate a scarf using automatically updated data for his or her hometown. Progress is tracked with a simple click or tap, and remembered between sessions. Updated data from Dayton University is automatically added to the display when available, and the user can always tweak their configuration.
