Temperature Scarf - Project Spotlight

The Temperature Scarf project (also known as My Year In Temperatures) is a knitting aid designed to help users create a scarf from the knitting / crocheting site Ravelry. First posted by Kristen Cooper, the idea is to use local daily average temperatures as a color key, producing a scarf that reads like a daily temperature record of your hometown.

I saw the opportunity to create a project that could automatically display the correct temperatures and colors for each day. With this functionality, the user no longer has to manually look up temperatures and find the correct color by hand.

A simple color configuration.

After picking colors and assigning names for easy identification, users can generate a scarf pattern using automatically updated data for his or her hometown. Progress is tracked with a simple click or tap, and is remembered between sessions. Updated data from Dayton University is added to the pattern as soon as it is available, and the chosen colors and temperatures can be reconfigured at any time.
