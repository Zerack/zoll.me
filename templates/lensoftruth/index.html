What is it?

lensoftruth.js is a jQuery plugin designed to help you when you're debugging complex usage of data-* attributes. jQuery hides these in an internal cache by default, so they aren't visible when you examine the DOM at runtime.

But that can change.

jQuery - Where are my data-* attributes?.

<div id="myDiv" class="myClass">...</div>

jQuery with lensoftruth.js - Look at all that data!

<div id="myDiv" class="myClass" data-lot-demo="true" data-lot-otherdemo="38">...</div>

Wait, what?

The lens of truth has revealed the data attributes that jQuery was hiding - but don't worry, they are just there to help you debug. jQuery is still independently caching your data attributes, so your code will run just as it did before. Notice that the data attributes are prepended with data-lot-. All data attributes written to the DOM by this plugin will have this prepended to avoid collisions.

A few examples:

If we start with <div id="myDiv" data-order="1"></div>, and run...

  • $('#myDiv').data() <div id="myDiv" data=order="1"></div> Object { order=1 }
  • $('#myDiv').data('sort') <div id="myDiv" data-order="1"></div> undefined
  • $('#myDiv').data('sort', 'asc') <div id="myDiv" data-lot-sort="asc" "data-order="1"></div> Object[div#testDiv]

It just works

Just include lensoftruth.js anywhere in your document after jQuery has been included, and this will just work. lensoftruth.js has been tested in jQuery 1.9.1, and leverages jQuery.attr() for cross browser compatibility. It will even convert attributes that are complex objects to JSON if your browser supports it.
